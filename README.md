# AI-Evolver

> **让你的 AI 助手越用越聪明的开发模板与全局大脑 / Make your AI coding assistant smarter the more you use it.**

本项目是一个完全与 AI 助手（Cursor/Copilot等）及特定 IDE/编译器解耦的自进化开发套件。通过“道术分离”的地图与知识蒸馏机制，实现开发经验在任何地方、任何项目中的完美复用与进化。

This project is an IDE/AI-agnostic self-evolving development kit. It makes your AI smarter over time and allows developer experience to be reused across any project or IDE.

## 📂 项目结构 (Repository Structure)

- **[`AI_Template_Project`](file:///AI_Template_Project/)**：自进化 AI 自治项目开发脚手架模板。(Self-evolving project scaffold)
- **[`AI_Global_Brains`](file:///AI_Global_Brains/)**：跨项目 AI 全局大脑皮层，负责踩坑经验的收集与知识提纯。(Global cross-project brain cortex)

有关各部分的详细使用说明，请参阅其目录下的子 `README.md` 文件。
For detailed instructions on each component, please refer to the respective `README.md` in each folder.
